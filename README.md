# UPSDIAGDd

## Installing

```
sudo su -
cd /path/to/where/you/want/store/upsdiagd
git clone https://github.com/Mausy5043/upsdiagd.git
cd upsdiagd
./install.sh
./update.sh
```

## Hardware:
Raspberry Pi 1B connected to a UPS via USB cable.
